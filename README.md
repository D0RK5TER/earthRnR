## Available Features (CRUD)
#
>Create
>- Create a review
>- Create a spot
>- Create a spot image

>Read
>- Read a spot and its images
>- Read the reviews
>- Read the session user's spots and reviews
>- Read spots with search parameters (pagination)

>Update
>- Update a review
>- Update a spot

>Delete
>- Delete a review
>- Delete a spot
